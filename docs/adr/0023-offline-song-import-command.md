# Offline Song Import Command

**Song Import** will start as an offline maintenance command run on the Windows machine that maintains the **Song Asset Directory**, rather than as a **Server** runtime feature exposed through **Master** or **Slave**. Importing downloaded source files, converting lyrics, and separating accompaniment are slow local file-maintenance work; keeping them outside the live **KTV Room** preserves the existing room boundary, and imported songs become available through the existing **Refresh Song Library** flow.
