# Demucs for Accompaniment Separation

The first **Song Import** implementation will call Demucs as an external offline tool to derive **Accompaniment** from decoded **Original Vocal** audio. Demucs is heavier to install and may be slow on CPU-only Windows machines, but it gives a better first quality baseline than older separation tools, and keeping it outside the live **Server** preserves the **KTV Room** runtime boundary.
